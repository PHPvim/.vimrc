# .vimrc

## Install

### Step by step
- First step: vim-plug, see https://github.com/junegunn/vim-plug#installation
- Second: start up vim/nvim. run `:PlugInstall`
- Third: reboot your vim/nvim

### One-script mode
```bash
curl -s https://git.io/ide | sh
```
